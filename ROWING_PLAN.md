## 📅 Les 3 phases

### **Phase 1 : Adaptation (W1-4)**

| | |
|---|---|
| **Fréquence** | 1×/semaine (mardi matin) |
| **Durée** | 30-40 min |
| **Intensité** | Zone 2 (60-70% FCmax) |
| **Focus** | Technique, stabilisation HRV |
| **Charge** | ~25 TSS/séance |

**Points clés :**
- Dos droit, jambes compression en dernier
- Respiration régulière (4-4 ou 6-6)
- Pas de fatigue excessive (HRV doit rester stable ±1-2 ms)
- Core engagé mais sans tension lombaire

**Sortie après phase 1 :** 100-160 min de rameur, ~100 TSS cumulés

---

### **Phase 2 : Consolidation (W5-12)**

| | |
|---|---|
| **Fréquence** | 1×/semaine (mardi 45-60 min) |
| **+ Optionnel** | Jeudi 40 min Z2-Z3 (si 2/sem souhaité) |
| **Intensité** | Zone 2 dominant, Z3 progressif |
| **Focus** | Capacité aérobie, force endurance |
| **Charge** | ~35-40 TSS/séance mardi |

**Progression :**
- Semaines 5-8 : Rallonger durée de 5 min/semaine jusqu'à 50 min
- Semaines 9-12 : Ajouter 1 intervalle Z3 (5-10 min) dans les séances

**Sortie phase 2 :** 450-500 min de rameur 8 semaines, ~280-320 TSS

---

### **Phase 3 : Progression (W13+)**

| | |
|---|---|
| **Fréquence** | 1-2×/semaine (selon TSB) |
| **Durée** | 50-70 min |
| **Intensité** | Mix Z2/Z3/Z4 |
| **Focus** | Force, capacité lactate, puissance |
| **Charge** | Variable 30-45 TSS/séance |

**À évaluer après phase 2 :**
- ✅ HRV stable ou en hausse ? → passer à 2 séances/semaine
- ⚠️ HRV en baisse (-3 ms+) ? → rester 1/semaine, réduire intensité
- 🔴 TSB < -15 ou fatigue ? → réduire volume, augmenter récupération

## ⏰ Micro-cycle recommandé

```
LUNDI      : Repos ou très léger (8-10 min Z1 marche)
MARDI      : 🚣 RAMEUR 45 min Z2 — MATIN 6h30-7h30 ← CLÉ
             Petit déj après
MERCREDI   : Course 5-8 km Z2 (léger)
JEUDI      : Repos
VENDREDI   : Optionnel — 🚣 RAMEUR 40 min Z2-Z3 (2/sem uniquement)
SAMEDI     : Course + tempo 30 min (seuil)
DIMANCHE   : Repos actif (marche, stretching)
```

**Pourquoi matin ?**
- Core "frais" = meilleure technique
- Évite fatigue lombaire en fin de journée
- Boost énergie pour le reste de la journée
- Intègre naturellement avant travail

## 🚨 Signaux de surcharge à monitorer

| Signal | Action |
|--------|--------|
| HRV baisse > 4 ms / semaine | ↓ durée -5 min ou skip jeudi |
| TSB < -20 | Pause rameur 1-2 semaines |
| Body Battery max < 40% | ↓ intensité Z2 pure |
| Sommeil < 7h pendant 3j | Skip séance, repos actif |
| Pace/vitesse running baisse | ↓ charge rameur ou volume |
| Sensation lombaire | Stop rameur, vérifier technique |

## 💡 Notes d'intégration

**Rameur ≠ Course :**
- Rameur = 85% muscles vs 40% running
- Plus grand stimulus systémique
- Récupération peut être plus longue les jours 2-3

**Compatibilité avec plan actuel :**
- Running : continue TTT
- Vélo : pas de conflit
- Natation : garder pour semaines de décongestion
- Strength : spacing de 48h si possible

**Progression type Z2 rameur :**

```
W1 : 3'00-3'10 par 500m (easy, pas essoufflant)
W4 : 2'50-3'00 par 500m (confortable)
W8 : 2'40-2'50 par 500m (fréquence ↑, puissance stabil)
W16 : 2'30-2'45 par 500m (plus reactif)
```
